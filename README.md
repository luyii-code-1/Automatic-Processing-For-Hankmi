# Automatic-Processing-For-Hankmi
## Hankmi商店源自动化制作
# AddJson.py
## 使用方法：
### 将所有的 APK 文件放入要执行脚本的目录中。
### 运行脚本，按照提示输入原始 JSON 文件的路径。
### 脚本将会自动将目录下所有 APK 文件名（不带后缀名）添加到用户提供的 JSON 文件中，并生成一个新的 JSON 文件。
# Auto.py
### 如果你的apk名称不规则，则需要使用此脚本批量重命名
### 将所有的 APK 文件放入要执行脚本的目录中。
### 运行脚本，重命名格式：应用名-版本
### 重命名失败的apk将移动至failed文件夹
